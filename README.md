# KANNet — ReLU Kolmogorov Arnold Network module for CNNs
### Personal Research Project
- Designed a custom PyTorch module replacing the standard Conv2d layer with a Kolmogorov–Arnold Network (KAN)-based mapping using ReLU activations.
- Leveraging the Kolmogorov–Arnold representation theorem, the module can theoretically learn any continuous mapping, extending the expressive capacity of convolutional networks.
- Implemented the KAN layer as a drop-in replacement within CNN architectures toevaluate empirical benefits and generalization
